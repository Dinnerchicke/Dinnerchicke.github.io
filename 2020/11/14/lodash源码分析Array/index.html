Array

chunk

._chunk(array,[size=1])

将array拆分成多个size大小的数组，如果最后的不够切则剩下的自己组成一个数组

compact

过滤原数组中的所有非假值元素。例如false, null, 0, "", undefined, 和 NaN 都是被认为是“假值”。

concat

让原数组与任何数组或值连接到一起
